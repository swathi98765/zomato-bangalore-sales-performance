# Zomato Bangalore Sales Analysis Dashboard

## Project Description
This project analyzes 7100+ restaurants in Bangalore to identify top-performing areas, restaurants, and cuisines.  
It includes:

- Feature engineering of orders, revenue, profit, and profit margin
- SQL analysis to find top restaurants, area-wise revenue, and top cuisines
- Interactive Power BI dashboard to visualize insights

## Skills Used
Excel | SQL | Python | Power BI


## Key Business Insights
- Koramangala and Indiranagar generate the highest revenue.
- Quick Bites restaurants have high profit margins.
- North Indian and Chinese cuisines dominate revenue generation.
- Fine Dining restaurants have lower ratings than casual dining.
- Whitefield is a high-revenue area despite fewer restaurants.
